# 🏨 LuxeStay Hotel Website

A premium hospitality website designed to showcase hotel services and improve booking engagement.

## 🚀 Features
- Multi-page layout (Home, Rooms, Services, Contact)
- Elegant and modern UI design
- Mobile responsive
- User-focused navigation

## 🎯 Purpose
Built to help hotels present their brand professionally and increase customer conversions.
